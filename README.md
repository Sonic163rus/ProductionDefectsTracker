# Production Defects Tracker
PWA-система для учета производственных дефектов с QR-интеграцией и офлайн-режимом

## Цель
Цифровизация контроля качества на производстве с минимальными затратами

## Стек
- Frontend: Vue.js PWA
- Backend: FastAPI
- База данных: SQLite/PostgreSQL
- Инфраструктура: Docker, ZeroTier

## Пилотное внедрение
Участок метизов без конвейера (завод, ноябрь 2025)
